# angular-md-gallery

AngularJS Material Design gallery module.

### Tech
This project is made possible with followings:
- CoffeeScript
- Jade
- Stylus
- Angular.js
- Angular Material
- Gulp
- And many other cool stuff.


### Development
#### Dependencies
```bash
 $ npm install
```

#### Run
```$
 $ npm start
```
